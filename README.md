# Página de Login - React + TypeScript

Uma página de login moderna com tema escuro desenvolvida em React e TypeScript.

## Tecnologias

- React 18
- TypeScript
- CSS3

## Funcionalidades

- Design dark mode
- Formulário de login com validação
- Componentes funcionais
- Interface responsiva

## Como executar

```bash
npm install

npm start
```
